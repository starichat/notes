# 该目录主要将概述程序设计的算法

程序=数据结构+算法

## 数据结构
数据结构的存储方式只有两种：数组和链表
在此基础上衍生出队列，堆，栈。。。都只是在数组和链表的基础上进行结合一些算法进行开发。


## 数据结构基本操作
任何数据的操作，基本操作无非就是遍历+访问。再具体一点就是增删改查。然后呢，数据结构其实有很多种类，有不同的应用场景，所以不同的数据结构的增删改查一般都是不同的。

线性：迭代
非线性：递归
